Foris forwarder
===============
Forwards Foris MQTT messages from/to subordinate message bus. It is also capable of discovering subordinates in the network using Zeroconf.

Requirements
============

* python3
* pyuci
* zeroconf
* paho-mqtt

Installation
============

	``pip install .``
